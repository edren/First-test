Release notes for Kendo UI v2013.1.319

Framework

What's new

  • Include VSDOC for all suites in Kendo Complete and Kendo Complete for ASP.NET MVC
  • Remove the default JSON parameterMap from DataSource Java wrapper implementation

What's fixed

  • Flip and pageTurn transitons do not work in IE10

DataSource

What's new

  • Support for virtualization with grouping

What's fixed

  • OData tranposrt does not support filtering with "Does not contain" operator

Globalization

What's new

  • Expand culture's numberFormat in order to support globalize.js
  • Allow to define shortYearCutoff option through culture info script.

What's fixed

  • kendo.toString hangs out when try to format number with "# yr." format
  • Numeric formatting rounds to the last zero placeholder in the decimal part incorrectly
  • kendo.parseDate parses incorrectly ISO8601 dates with timezones

Grid

What's fixed

  • Sorting options of the column menu does not work on touch devices

HierarchicalDataSource

What's new

  • Breaking change: children datasources are now lazily initialized

MVVM

What's new

  • Support for setting <select> values which do not exist in the source binding.

What's fixed

  • MVVM clears the widget, which initiated the model change
  • Multi select selection updated if an item is deselected

Validator

What's new

  • Method for hiding all validation messages

Mobile

What's new

  • useNative option for Scroller and useNativeScrolling option for Mobile View
  • Support for server-side navigation for remote views
  • Improve view transitions on mobile devices (sometimes they stop abruptly)
  • on remote view load, If no data-role=view element is found contents are treated as a single view
  • View afterShow event (triggered after the transition is complete)
  • Add init, open and close events for the Modal View widget
  • Visual Studio 2012 intellisense annotations
  • TypeScript support
  • RequireJS support
  • Improve remote view with parameters behavior
  • Introduce contentElement() method to always get the contents parent
  • Allow scrollview to scroll to percentage of its width
  • ButtonGroup touchstart/end configuration option
  • Add support for datetime-local input type.

What's fixed

  • ScrollView does not work inside a Modal View
  • Nexus 10 is not recognized as a tablet
  • Textarea cannot be scrolled when inside a mobile application view
  • 2 actionsheets do not work in view with a model
  • ScrollView does not retain its offset on orientation change when pages are present
  • Browser window can be dragged on iOS during View transition.
  • View Title gets bigger if it has block content
  • ListView endless scrolling does not work in a modalview.
  • ScrollView does not set its page correctly via data attribute

DropDownList

What's fixed

  • DropDownList gets opened when being a start point for scrolling in Kendo UI Mobile

ListView

What's new

  • Add/Remove items depending on the applied action to the observable array.
  • ListView filtering

What's fixed

  • Wrong property name in Kendo Mobile Listview
  • Mobile Scroller stops working when user has a finger on the ListView while DataSource changes
  • Grouped widget throws a JavaScript exception when clear data using query method

Scroller

What's fixed

  • Mobile Scroller stops working when user has a finger on the ListView while DataSource changes

SplitView

What's fixed

  • Kendo mobile pane does not support initial view setting (docs have it)

TabStrip

What's fixed

  • TabStrip with URL parameters doesn't reflect the view change on app.navigate()
